Git Ignore

Paso 1: Crea o modifica el archivo .gitignore en la raíz de tu proyecto:

touch .gitignore

Paso 2: Abre el archivo en tu editor preferido, como:

nano .gitignore

Añade reglas para evitar seguimiento de archivos como imágenes o videos:

# Imágenes
*.jpg
*.jpeg
*.png
*.gif
*.svg

# Videos
*.mp4
*.avi
*.mov
*.mkv

# Audios
*.mp3
*.wav
*.flac

Paso 3: Asegúrate de que Git ignore estos archivos no agregados. Si ya fueron añadidos, elimínalos del seguimiento sin borrarlos del sistema:

git rm --cached *.jpg
git rm --cached *.mp3

Paso 4: Confirma los cambios y prueba que el .gitignore funcione correctamente:

git add .gitignore
git commit -m "Ignorar archivos de imágenes y videos"
